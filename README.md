# SimpleRouting PHP
A simple PHP single file routing<br />
Do not need any dependence, based on network code modification
Parameter passing is not supported and is only used as a simple route
 
***
 
## How to use it?
You need to quote 'routing.php' in the entry file

## How to modify?
Please add this template in turn under the case statement<br />
`case "URI path":      include __DIR__ . "File path";  break;`
